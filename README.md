# 🌱 AI & IoT-Based Smart Agriculture System

An open-source intelligent smart agriculture platform that combines **Artificial Intelligence (AI)**, the **Internet of Things (IoT)**, and **embedded systems** to monitor environmental and soil conditions in real time and recommend the most suitable crops using machine learning.

## 📖 Overview

Modern agriculture requires accurate monitoring and intelligent decision-making to maximize crop productivity while reducing water and fertilizer consumption. This project provides an affordable and scalable smart farming solution by integrating environmental sensors, embedded hardware, and AI-based crop prediction.

The system continuously collects data from multiple sensors, processes the information, and displays it through an intuitive desktop application. A machine learning model analyzes the collected data and recommends the most suitable crop based on current environmental and soil conditions.

---

## 🚀 Features

* 🌡️ Real-time monitoring of environmental and soil conditions
* 🌱 AI-powered crop recommendation system
* 📊 Live sensor visualization through a graphical user interface
* 💾 Automatic data logging and storage
* 📈 Historical data analysis
* 🔌 Serial communication with embedded hardware
* 🌍 Low-cost and scalable architecture
* 🧠 Machine Learning integration for intelligent decision making

---

## 🔬 Sensors Used

The system supports monitoring of:

* Soil Moisture
* Soil Temperature
* Air Temperature
* Air Humidity
* Soil pH
* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)

Additional sensors can easily be integrated.

---

## 🤖 Artificial Intelligence

A machine learning model was trained using agricultural datasets containing environmental and soil parameters.

Based on the collected sensor data, the model predicts the most suitable crop for cultivation under the current conditions.

This enables farmers to make data-driven decisions and improve agricultural productivity.
